### Phylogeny

Change identifiers of fasta file to avoid problems with the different programs:

```
sed 's/,/_/g' ACT1_Exon2_for_phylogeny.fasta > ACT1_Exon2_for_phylogeny_nocomas.fasta
sed 's/ /_/g' ACT1_Exon2_for_phylogeny_nocomas.fasta > ACT1_Exon2_for_phylogeny_nocomasnospaces.fasta
sed 's/:/_/g' ACT1_Exon2_for_phylogeny_nocomasnospaces.fasta > ACT1_Exon2_for_phylogeny_nocomasnospacesnocol.fasta
sed 's/-/_/g' ACT1_Exon2_for_phylogeny_nocomasnospacesnocol.fasta > ACT1_Exon2_for_phylogeny_nocomasnospacesnocolnolines.fasta
```

Also, the beginning of some names were changed because PhyML sometimes only shows 10 characters.

##### Muscle

```
muscle -in ACT1_Exon2_for_phylogeny_nocomasnospacesnocolnolines.fasta -out Muscle_ACT1_Exon2_for_phylogeny.fasta
```
